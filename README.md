# Deep Learning with PyTorch 🧠

Complete deep learning implementation from 
scratch using PyTorch — neural networks to transformers!

## Projects Inside

### 1. Neural Network
- 3 layer network for student pass/fail prediction
- BCELoss + Adam optimizer
- Forward pass + backpropagation

### 2. CNN on MNIST ⭐
- Convolutional Neural Network
- Trained on 60,000 handwritten digit images
- **98.74% accuracy on 10,000 test images!**
- Matches human level performance!

### 3. RNN Sequence Predictor
- Recurrent Neural Network
- Predicts next number in sequence
- Normalized data + MSELoss

### 4. LSTM
- Long Short Term Memory network
- Better memory than RNN
- Handles longer sequences!

### 5. Transformer & Attention
- Self attention from scratch!
- Multi head attention mechanism
- Layer normalization + feed forward

### 6. HuggingFace Pipelines
- Sentiment analysis (100% accurate!)
- GPT2 text generation
- Zero shot classification (96.2%!)
- BERT fill mask

### 7. Fine Tuned BERT
- Custom sentiment classifier
- Trained on own dataset
- 100% test accuracy!

### 8. LangChain + Groq
- LangChain prompt templates
- Groq API integration
- Real Llama3 AI answers!

## Tech Stack
- PyTorch
